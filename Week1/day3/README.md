# Day 3 - Understanding Tokens in Large Language Models

## 📌 Objective

Learn what tokens are, why they are important in Large Language Models (LLMs), and how they affect API usage, response length, and pricing.

---

# 🧠 What are Tokens?

A token is the smallest unit of text that an LLM processes. Instead of reading an entire sentence at once, the model breaks the text into smaller pieces called **tokens**.

A token can be:

- A complete word
- Part of a word
- A punctuation mark
- A number
- A special symbol

For example:

```
Hello, world!
```

may be divided into multiple tokens depending on the tokenizer used by the model.

---

# ❓ Why are Tokens Required?

LLMs do not understand text the way humans do. Before processing any input, the model converts text into tokens.

Without tokenization, the model cannot:

- Read user input
- Understand context
- Generate responses
- Calculate computational cost

Tokens act as the language that the model actually understands.

---

# 📍 Where are Tokens Used?

Tokens are involved in every stage of an LLM interaction.

### User Prompt

The text entered by the user is converted into **prompt tokens**.

### Model Response

The generated answer is counted as **completion tokens**.

### Total Usage

The total number of tokens processed is:

```
Total Tokens = Prompt Tokens + Completion Tokens
```

---

# 💰 Why are Tokens Important?

Tokens are important because they determine:

- API usage
- Response length
- Processing time
- Cost of using an LLM

Most AI providers charge based on the number of tokens processed rather than the number of requests.

---

# 📊 Types of Tokens

### Prompt Tokens

Tokens present in the input provided by the user.

### Completion Tokens

Tokens generated by the AI in its response.

### Total Tokens

The sum of prompt tokens and completion tokens.
